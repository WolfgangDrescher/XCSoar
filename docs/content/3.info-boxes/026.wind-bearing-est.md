Wind bearing estimated by XCSoar. The secondary value shows the wind speed.
Manual adjustment is possible via the InfoBox dialogue: up/down cursor keys
cycle through settings and left/right cursor keys adjust values. Clicking the
InfoBox opens the wind adjustment dialogue.
