The navigation altitude used by XCSoar. The title dynamically changes: shows
"Alt Baro" when barometric altitude is used (QNH set and baro altitude available
and nav baro altitude enabled in settings), or "Alt GPS" when GPS altitude is
used. Shows "no QNH" as comment when pressure altitude is available but QNH is
not set. An InfoBox dialogue is available.
