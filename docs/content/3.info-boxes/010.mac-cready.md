Current MacCready setting. The title dynamically changes to "MC AUTO" when
MacCready is in automatic mode, or "MC MANUAL" when in manual mode. The value
color changes accordingly: green in auto mode, blue in manual mode. Pressing the
Enter cursor key toggles Auto MacCready mode. When active (Touch-screen/PC
only), the up/down cursor keys adjust the MacCready setting. An InfoBox dialogue
is available.
