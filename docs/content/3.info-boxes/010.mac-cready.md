Current MacCready setting, current MacCready mode (manual or auto), and
recommended speed-to-fly. (Touch-screen/PC only) Also used to adjust the
MacCready setting if the InfoBox is active, by using the up/down cursor keys.
Pressing the enter cursor key toggles `Auto MacCready' mode. An InfoBox dialogue
is available
