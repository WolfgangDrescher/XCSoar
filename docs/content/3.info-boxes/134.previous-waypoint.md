Previous task waypoint: when an ordered task is loaded, automatically tracks the
waypoint before the active leg (the start when on the first leg). The title
dynamically shows the waypoint name, or "Prev WP" when none. Displays the
waypoint name, arrival altitude difference, and distance. Clicking opens a
dialogue to choose a different waypoint to display (informational only — never
advances the task or sets a Goto). With a task loaded, choose "Resume auto
tracking" at the top of the list to revert to automatic tracking.
