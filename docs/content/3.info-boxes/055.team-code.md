Current team code for this aircraft, used to report position to other team
members. The secondary value shows the last entered team aircraft code, or the
FLARM callsign if a teammate is tracked via FLARM. Clicking the InfoBox opens
the team code entry dialogue. When active, cursor keys adjust the team code.
