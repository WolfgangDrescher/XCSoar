Instantaneous MacCready dolphin speed-to-fly, using netto vario calculations for
the current track. In cruise mode it is calculated for maintaining altitude; in
final glide mode for descent; in climb mode it shows the speed for minimum sink
at the current load factor (if an accelerometer is connected). The secondary
value shows "BLOCK" when Block mode speed-to-fly is selected, or "DOLPHIN" when
dolphin mode is active. When Block mode is selected, this InfoBox shows the
MacCready cruise speed.
