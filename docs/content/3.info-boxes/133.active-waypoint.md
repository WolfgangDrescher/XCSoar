Active task waypoint: shows the next task waypoint when an ordered task is
loaded, or the Goto waypoint otherwise. The title dynamically shows the waypoint
name, or "Active WP" when no waypoint is active. Displays the waypoint name,
arrival altitude difference, and distance. Clicking opens a dialogue to choose a
different waypoint: for a task, selecting a task waypoint skips to that leg;
with no task loaded, any waypoint sets it as the Goto destination.
