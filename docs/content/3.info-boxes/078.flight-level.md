Pressure altitude expressed as a Flight Level (FL). When derived from barometric
(pressure) altitude, the title label is shown in black. When calculated from GPS
altitude (because barometric altitude is unavailable but QNH is set), the title
label turns red to indicate reduced accuracy. The secondary value shows the
altitude in feet. Displays `---` with comment "no QNH" when neither GPS altitude
nor QNH is available. An InfoBox dialogue with Simulator, Info, and Setup panels
is available.
