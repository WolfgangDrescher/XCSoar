Time-based start gate status showing the signed countdown until the start gate
opens or closes. The value color and secondary line reflect the gate state: blue
with "Too early" when waiting for the start window, green with "Can start" when
the window is currently open, and red with "Too late" when it has closed. If the
PEV Start Procedure is in use, time relative to the PEV window is displayed;
otherwise the task-defined start open/close times are used. Clicking opens the
start details dialogue.
