Arrival time to start with start gate status. The main value shows estimated
time to reach the start line (MacCready leg when available, else distance
divided by ground speed). The title dynamically changes based on state: "Start
reach" when arrival is predictable, "Start unknown" when unpredictable, "Start
open" when the gate is currently open. The value color and secondary line
reflect the gate state at that estimated arrival time: blue when too early,
green when the gate will be open, and red when too late. If the PEV Start
Procedure is in use, time relative to the PEV window is displayed; otherwise the
task-defined start open/close times are used. Clicking opens the start details
dialogue.
