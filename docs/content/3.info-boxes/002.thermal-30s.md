30-second rolling average climb rate based on GPS altitude, or vario if
available. The value turns red when the 30-second average is less than half the
current risk MacCready value, indicating the climb is too weak relative to the
calculated risk. The secondary value (in smaller font) shows the climb rate for
the current thermal since circling started.
