Current QNH pressure setting used for barometric altitude calculations.
Displayed in the current pressure unit (hPa or inHg). Displays `---` with
comment "no QNH" when QNH has not been set. Tapping the InfoBox opens the
altitude setup panel to adjust QNH manually.
